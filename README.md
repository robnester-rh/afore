# afore
### A command-line utility to trigger testing

## Installation

### With `go get`

```bash
go get github.com/robnester-rh/afore
```

### Direct download

```bash
wget
``` 

### Building locally

```bash
$ git clone https://github.com/robnester-rh/afore.git
$ cd afore
$ make release
$ cd release
$ ./afore-<version>-<os>-amd64
```
